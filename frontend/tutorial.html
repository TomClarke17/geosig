Admin Tutorial

This guide provides step-by-step instructions for administrators using the GeoSig admin panel.

Managing QR Codes

  • Log into the Admin Panel.
  • Navigate to **QR Code Management**.
  • Click **+ Add QR Code** to register a new QR code.
  • Enter necessary details such as **location and metadata**.
  • Click **Save** to confirm.

Verifying Users

  • Go to **User Management**.
  • View pending verification requests.
  • Approve or reject users based on credentials.
  • Users will be notified of their verification status.

Security Best Practices

  • Regularly audit registered QR codes.
  • Ensure that only **verified admins** have access.
  • Use **strong passwords** for admin logins.

User Tutorial

This guide provides step-by-step instructions for users accessing the GeoSig app.

Scanning QR Codes

  • Open the **GeoSig app**.
  • Tap **Scan QR Code**.
  • Allow **camera access** if prompted.
  • Scan the QR code and follow the on-screen instructions.

Viewing Past Scans

  • Go to **History** in the app menu.
  • View all previously scanned locations.

Managing Account

  • Go to **Settings** to update account details.
  • Enable **notifications** for scan alerts.
